Melodic_Steps_Crop.png

Melodic Stepsは展開するメロディをわずか数クリックで生成できる素早く直感的なMIDIシーケンサーです。バリエーションやポリメトリックパターンの作成、あるいは作成内容のランダマイズに使用します。

インターフェースはプレイ向けにデザインされており、実験的操作や新たな発見を促進します。ステップに数値は表示されません。代わりに、数値により音楽を作成するのではなく、そのコントロールを使用して直感的にプレイできます。

ステップは、クリック&ドラッグで直接、または各タイムラインで使用できるランダムとリセットのボタンを使用して微調整できます。この機能で特徴的なのは、反復のオーバーラップの仕方によって各反復の聞こえ方がわずかに異なるよう、さまざまなタイムライン長さ（2～16ステップ）をずらすことができる点です。タイムラインは再同期ボタンでいつでも再同期させることができます。

シーケンサーは通常トランスポートに追従します。また、MIDIノート入力はリアルタイムでルートを変更するのに使用できます。MIDIボタンが有効な場合、シーケンサーは、ノートメッセージが押されている間だけタイムラインを再生するため、デバイスはベーシックなアルペジエーターとして機能します。

タイムラインコントロール：

-> Octave: ステップを+/-3オクターブでトランスポーズ
-> Transpose: ［Scale & Root］セレクターで選択されているスケールに沿ってステップをトランスポーズ
-> Length: ステップの長さ
-> Velocity: ステップのベロシティ
-> Chance: 再生されるステップのチャンスを設定

Pitch Hackはピッチ関連のさまざまなエフェクトをもたらすワンストップデバイスです。Pitch Hackは半音とセントの両単位のトランスポーズコントロールを装備した単体のディレイラインです。受信する信号のリアルタイムでのピッチ変更に加えて、Pitch Hackはオーディオの反転、トランスポーズのインターバルのランダマイズ、ディレイ信号のディレイラインへの畳み込みが行えます。

-> インターバル: ディレイ部分の半音単位のトランスポーズ
-> Fine: ディレイ部分のセント単位の細かなトランスポーズ
-> Var: インターバルトランスポーズのランダムなバリエーション。0からこのパラメーター間のランダムな値は各トランスポーズ部分に対して選択（下参照）
-> Rate: トランスポーズ済み信号の部分の長さ
-> Reverse: 部分が反転再生される確率
-> Recycle: ディレイラインに戻されるトランスポーズ済み信号の量
-> Level: ディレイラインのゲイン

Gated_Delay.png

Gated Delayはシーケンスされたディレイエフェクトを作成します。ゲートシーケンサーは受信信号をアクティブなステップのディレイラインへと送信します。定義されたリズムで音楽に合わせてオンとオフが切り替わるセンドエフェクトのようなものです。

コントロールでは、ステップの長さとレート、アクティブでないステップでのドライ信号の可聴の可否、ディレイレート対ステップレートの比率、ステップのランダム化、フィードバックなどを調整できます。

-> Steps: ゲートシーケンスの長さ
-> Rate: ゲートシーケンスのレート（音価単位）
-> Smooth: ゲートが開く/閉じる際のフェードイン/フェードアウト時間（ミリ秒単位）
-> Delay: ステップ長さに対するディレイ時間（相対値）。1の場合ディレイはステップと同じ長さ、2の場合ステップの2倍、1/2の場合半分
-> Random: ディレイ時間のランダムなバリエーション
-> Feedback: フィードバックアマウント
-> Insert/Gateトグル: ［Insert］モードの場合ゲートがオフの際はドライ出力が有効、［Gate］モードの場合ゲートがオフの際はドライ出力がミュート
-> Delayのボリューム: ディレイラインのゲイン

Color Limiterはハードウェアリミッターの粗さのあるサウンドにインスピレーションを受けた独特の味のあるリミッティングを提供します。入力レベル（Loudness）、スレッショルド（Ceiling）、Lookahead、Release、そしてその独特なサウンドに重要となるSaturation、Colorパラメーターをコントロールできます。

-> Loudness: 入力レベル
-> Ceiling: スレッショルドレベル
-> Release: 入力サウンドがスレッショルド（Ceiling）を下回る際にリミッターがフル振幅に戻るのに必要な時間を設定
-> Saturation: 出力ボリュームをさらにブーストしてディストーションを生成
-> Color: サチュレーションの周波数特性。値が低めだとよりダークなサウンドに
-> Lookahead: リミッターがサウンドの軌道を事前に把握して適切に振幅を変更するのに与えられる時間を設定

Re-Enveloper.png

Re-Enveloperはマルチバンドエンベローププロセッサーです。入力信号はまず調整可能な3つの周波数帯に分割されます。その後、エンベロープが各帯域に適用されます。これには圧縮や拡張が可能です。

-> Freq1: 周波数を3帯域に分割します。低域が0Hz～Freq1、中域がFreq1～Freq2です
-> Freq2: 周波数を3帯域に分割します。中域がFreq1～Freq2、高域がFreq2より上です

各帯域のパラメーター：

-> オン/オフ: 各帯域のオンとオフを切り替えます
-> Vol: ボリューム
-> Attack: エンベロープが上行の場合の反応
-> Release: エンベロープが下行の場合の反応
-> C/E Fact: コンプレッサー/エキスパンダー係数。正の値の場合エンベロープは圧縮され（ソフトな部分とラウドな部分の差異が小さくなる）、負の値の場合エンベロープは拡張され（ソフトな部分とラウドな部分の差異が大きくなる）ます

Spectral_Blur.png

Spectral Blurはリバーブのようなサウンドとテクスチャを作成します。ユーザーが定義する信号の周波数範囲にブラーがかかり、もやっぽいサウンドを作成できます。このデバイスは、狭い帯域幅のシャープなフィルターエフェクトやサウンドをスペクトル全体にわたって分散させるのに最適です。

-> In/Outスイッチ: ［In］位置の場合、Freq1とFreq2の間の信号が処理されます。［Out］位置の場合、Freq1とFreq2の外側の信号が処理されます
-> Freq1、Freq2: ［In］/［Out］スイッチの設定に基づいてブラーがかかった信号の周波数範囲を設定します
-> Residual: 未処理成分の振幅
-> Halo/Freezeスイッチ: ［Freeze］モードの場合、ブラーがかかった成分はフリーズ化され、［Halo］パラメーターは効果を持ちません
-> Halo: ブラーがかかったグレインの長さを設定します
-> Delay Comp: 処理済み信号との完璧な同期が必要な場合のドライ信号のディレイ補正
-> Wet Gain: エフェクト済みの信号のボリューム

注：Spectral Delayには処理済み信号と入力信号を調整するディレイ補正機能があります。ディレイ補正が1に設定されている場合、たとえばディレイ補正が1に設定されているSpectral Blurの後にあるパラメーターのオートメーションなど、一部の要素のタイミングがずれることがあります。

Bass_Crop.png

Bassは、ディープなクラシックから強烈なディストーションがかかったベースサウンドまで、幅広い音をもたらすバーチャルアナログのモノフォニックシンセです。

*オシレーターセクション*

4つの個別のバーチャルアナログオシレーターがあります。正弦波、のこぎり波、三角波、矩形波で、それぞれに個別のボリュームコントロールがあります。後者はさらにさまざまなパルス幅に対応しており、また全オシレーターは[Analog](アナログ)コントロールを使用することで互いにデチューンできます。[Tune](チューン)ボタンでは各オシレーターを+/-12半音でチューンできます。

[Suboscillator](サブオシレーター)は単一の正弦波ジェネレーターから構成されており、[Tone]パラメーターを使用して変形させて幅広い倍音スペクトルを作成できます。[Thru]トグルを使用して、サブオシレーターをフィルターセクションに通したりバイパスしたりできます。

Bassには[Hard Sync](ハード同期)と[Ring Modulation](リングモジュレーション)に専用のセクションが用意されており、パラメーターはオシレーターの下に表示されます。また、同期またはリングモジュレーションのピッチを調整するための振幅エンベロープ用の追加コントロールもそこに表示されます。[Sync]と[Ring Modulation]のチューニング周波数は単一のパラメーターを使用してさらに調整できます。

*フィルターセクション*

Bassには5種類のフィルタータイプが用意されています。[LP]と[LP2]は信号の変更方法を大幅に変化させ、どちらもユニークなサウンドシェイピング特性を提供します。すべてのフィルターには[Cutoff](カットオフ)と[Resonance](レゾナンス)パラメーターがあります。[Cutoff]は[Filter Env]ダイヤルを介してフィルターエンベロープを使用してモジュレートできます。[Drive](ドライブ)パラメーターはフィルターに入力される前の信号をオーバードライブします。

*モジュレーション*

LFOはいくつかの波形とノートが再生された際にそのアマウントをスムーズにフェードインするオプションの[Fade-in]パラメーターを提供します。

2つのエンベロープはノンリニア形状を使用します。[Attack](アタック)、[Decay](ディケイ)、[Release](リリース)の範囲は、[Env Time](エンベロープタイム)を使用して拡張できます。

*マスターセクション*

[Master](マスター)セクションには粗調整と微調整オプションが用意されており、ピッチを臨機応変に変更でき、粗調整から微分音デチューンまで行えます。グローバルの[Distortion](ディストーション)ノブはサチュレーションを加えます。

Poliは、コードやポリフォニックフレーズ向けにデザインされた、クラシックなサウンドをもたらすバーチャルアナログシンセです。その特徴はデチューン可能なオシレーター、多様なモジュレーションセクション、内蔵コーラスによるものです。

*オシレーターセクション*

オシレーターセクションには、5種類のサウンドジェネレーターと独立ボリュームコントロールが含まれています。[Saw]、[Pulse]、[Sub]オシレーターはバーチャルアナログオシレーターです。[Saw]オシレーターのピッチは[Pulse]オシレーター信号でモジュレートでき、[XMod]コントロールを使用して繊細なものから強烈なものまでバリエーションを作成できます。[Sub]もパルス波ですが、メインピッチの1または2オクターブ下にトランスポーズされ、リッチな低音を追加します。

[Ring]は[Pulse]オシレーターを指定の周波数の正弦波でリングモジュレーションすることで信号を作成するオシレーターです。周波数は[Freq]パラメーターと[Key]トラッキングアマウントを使用して調整できます。[Noise]ノブはミックスにホワイトノイズを加えます。

*フィルターセクション*

すべてのオシレーターはハイパスとローパスのフィルターに供給されます。ローパスには[Resonance]パラメーターがあり、フィルターのカットオフ周波数のパフォーマンスコントロールの[Key]、[Velocity]、[After-touch]トラッキングでモジュレートできます。

*モジュレーションセクション*

このセクションには、[LFO]、[Filter Env]、[Amp Env]の3つのソースのモジュレーションマトリックスと7つのターゲットがあります。マトリックス内のエントリをクリック&ドラッグしてモジュレーションをターゲットに適用します。

LFOは、フリーチューニング(ノートオン時のリトリガーあり/なし)またはLiveのトランスポートに同期するよう設定できます。いくつかの波形を選択でき、LFOレートはマトリックス内のフィルターエンベロープを使用してさらにモジュレートできます。

2つのエンベロープの形状はアナログかつノンリニアで、自然なレスポンスをもたらします。フィルターエンベロープにはさらに[Loop]ボタンがあり、エンベロープをLFOに変化させます。レートは[Attack]と[Decay]の設定により決定されます。

*マスターセクション*

内蔵[Chorus](コーラス)には3種類のモードが用意されています。モードAはソフトで繊細なエフェクト、Bはクラシックなストリングコーラスです。モードCはアグレッシブな「レイヴ」タイプのエフェクトで、独特の特徴があります。

[Random Pan]は各ノートが右または左チャンネルにランダムにパンされる量を調整し、クラシックなステレオ効果をもたらします。

[PBend]コントロールはピッチベンドホイールがシンセサイザーのパッチを変更する度合いを定義します。下のスイッチでピッチベンドがノコギリ波信号のみに影響するのかすべてのオシレーターに影響するのかを決定します。

マスターセクションにはポリフォニック[Glide]も含まれており、グライド時間を0～2秒の間で調整できます。また、[Poly]、[Mono]、[Unison]の間で切り替えることができ、スプレッドを調整できます。
